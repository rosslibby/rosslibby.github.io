Let's build an HTML website

Comments in HTML

Every programming language we will use has comments. Comments are human-readable text that is not interpreted as code. Comments often serve as notes that the programmer leaves to him- or her-self, or to inform other programmers what is going on in a certain part of the code. Comments are also incredibly useful for temporarily removing pieces of code that the programmer may wish to use at some point in the future, though it may not be currently relevant.

Comments in HTML are created by typing the following:

<!-- This is a comment. It will not be treated as code and it will not be displayed on the website. -->

Layout in HTML

Websites are divided into multiple sections within their HTML code, where
each section is styled in order to add form to the information.

We can lay out content in HTML using a couple of key elements:

  • Dividing containers <div>...</div>
  • Inline containers <span>...</span>

We use the <div> tag when we need to separate some elements from other elements, whether for layout or aesthetic purposes.
